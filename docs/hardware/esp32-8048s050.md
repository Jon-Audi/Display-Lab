# ESP32-8048S050

## Overview

The ESP32-8048S050 is a 5-inch RGB TFT display board using the ESP32-S3 microcontroller.

## MCU

ESP32-S3-WROOM-1-N16R8

Specifications:

| Feature | Value |
|------|------|
CPU | Dual-core Xtensa LX7 |
Clock | 240 MHz |
Flash | 16MB |
PSRAM | 8MB |

---

## Display

Resolution: 800 × 480

Color format:

RGB565

Framebuffer size:

```
800 × 480 × 2 bytes
= 768 KB
```

Double framebuffer:

1.5 MB

---

## Touch

Controller: GOODIX GT911

Interface: I²C

Pins:

SDA
SCL
INT
RST

---

## Peripherals

MicroSD card slot

USB-C for power/programming

Speaker output via amplifier

---

## Typical Use Cases

Smart home dashboards
Vehicle displays
Media controllers
Industrial HMI
