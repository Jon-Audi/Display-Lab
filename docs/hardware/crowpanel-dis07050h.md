# CrowPanel DIS07050H

## Overview

5-inch ESP32-S3 based HMI display module.

## Core Module

ESP32-S3-WROOM-1-N16R8

| Feature | Value |
|--------|------|
CPU | Dual core Xtensa LX7 |
Clock | 240 MHz |
Flash | 16MB |
PSRAM | 8MB |
Display | 800x480 RGB |
Touch | Capacitive |
Audio | Speaker output |
Storage | microSD |

## Display Interface

The display uses the ESP32 LCD_CAM RGB interface.

Signals include:

R0–R7
G0–G7
B0–B7
HSYNC
VSYNC
DE
PCLK
